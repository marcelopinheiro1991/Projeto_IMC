📌 Sobre o Projeto

O Sistema Web de Cálculo de IMC (Índice de Massa Corporal) foi desenvolvido com o objetivo de facilitar a avaliação da saúde corporal de usuários em ambientes como academias e projetos educacionais.

A aplicação permite calcular automaticamente o IMC a partir do peso e altura informados, exibindo a classificação do resultado de forma clara e intuitiva.

🎯 Objetivo
Automatizar o cálculo do IMC
Auxiliar alunos e professores em acompanhamento físico
Facilitar o registro e análise de dados em academias
Promover consciência sobre saúde e qualidade de vida
⚙️ Tecnologias Utilizadas
HTML5 → estrutura da página
CSS3 → estilização e layout
JavaScript → lógica do cálculo do IMC
🧮 Como Funciona

O sistema realiza o cálculo do IMC utilizando a fórmula:

IMC = peso / (altura × altura)

Após o cálculo, o sistema retorna a classificação:

Abaixo do peso
Peso normal
Sobrepeso
Obesidade
💻 Funcionalidades
Entrada de peso e altura
Cálculo automático do IMC
Exibição da classificação
Interface simples e responsiva
🌍 Impacto do Projeto

Este sistema contribui para a educação em saúde, ajudando usuários a entenderem melhor seu estado físico. Em ambientes acadêmicos e de academias, também auxilia no acompanhamento de alunos e clientes de forma prática e rápida.
